cosmo 紹介

cosmoとは、Silicon Graphics Inc.(SGI) 社製の O2 （オーツー）という UNIX ワークステーションです。このマシンは、図書館情報大学第2大講座（通称2講）に所属し、cosmo.ulis.ac.jp というホスト名を持っています。

cosmo.ulis.ac.jpは原則として、 本学の学生・教官およびその関係者より利用の申し出があれば、 随時 ID を登録しています。 利用されたい方はユーザ登録グループ(admin@nile.ulis.ac.jp)まで御連絡下さい。 ユーザ登録を行ないます。

何ができるか

一般にグラフィックス・ワークステーションと呼ばれるSGI社製のワークステーションなので、画像処理に優れているらしいです。さらに、このマシンは、WebFORCEモデルと呼ばれるモデルなので、デフォルトでWWW上でのコンテンツの作成・提供用に様々なソフトウェアがバンドルされています。以下、

  • WWWブラウザ：Netscape Navigator 3.01 Gold
  • WWWサーバ：Netscape FastTrack
  • HTML作成ツール：Cosmo Create
  • VRMLオーサリング環境：Cosmo Worlds
  • Java開発環境：Cosmo Code
  • VRMLブラウザ：Cosmo Player
  • 画像処理：Adobe Photoshop
  • Photoshop用プラグイン：Kai's Power Tools
  • 描画ツール：Adobe Illustrator
などです。

その他の機能としては、

  • オペレーティング・システム：IRIX 6.3
  • ネットワーク：TCP/IP、Netware, Xinet, AppleTalk, XFS
  • ビデオ入出力
  • OpenGL開発環境
  • Windowsサポート：SoftWindows 2.0
  • 日本語入力環境：Wnn6
などがあります。

その他の詳細な説明などは、SGI社のサイト（http://www.sgi.co.jp/Product/WebFORCE/products/wfo2.html）などを御覧下さい。
