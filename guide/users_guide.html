cosmoにおける自分のわかる範囲での、簡単なユーザーズガイドです。 詳しい情報に関しては、ヘルプやオンラインマニュアルを参照下さい。

基本

まず、基本的に利用するコマンドなどです。
エディタ
vi
UNIX 標準のスクリーンエディタです。
mule
GNU Emacs の多言語対応エディタです。
jot 及び nedit
デフォルトでついてきた GUI のテキストエディタです。
日本語入力環境
良くわからないけどWnn6です。通常は、xwnmo2が動きます。
日本語入力は Control + バックスラッシュで出来るようになります。
変換はスペースキーです。
文節移動は Control + f（前）、Control + b（後）。
文節の伸縮は、Control + o（伸）、Control + i（縮）。
こんなもんで多分一応は使えるでしょう。
印刷
lpコマンドです。（lpr は使えません）利用できるプリンタは、600ps（デフォルト）, b406ps の2つだけです。これ以外のプリンタへは、rsh などで他のマシンを通して下さい。 
       600psに出す場合：  % lp ファイル名
       b406psに出す場合： % lp -d b406ps ファイル名
       または、
       % cat ファイル名 | rsh 他のマシン名 lp -d 利用したいプリンタ名
*日本語テキストファイルの出力については、文字化けする場合があります。a2ps コマンドを通して下さい。(% a2ps ファイル名 | lp )
WWWブラウザ
SGI版 Netscape Navigator 3.01 GOLD が利用できます。コマンドは netscape です。
リモートでの作業
cosmo から他のマシン（汎用SUN）などで rlogin などで作業をする際、シェル変数の term の初期化がうまくできないと、画面が乱れます。画面が乱れた場合は、 
       % setenv TERM ansi
       または
       % set term=ansi
       として下さい。
       また、汎用・共用 SUN では、ログイン時にTERMを以下のように聞かれた場合は、
       Type iris-ansi-net unknown
       TERM = (unknown) 
       ansi と入力して下さい。
プログラム開発環境
  • C言語
    cc コマンドで使えると思います。gcc コマンドもあります。
  • Java言語
    JDK1.1.2相当のものがあります。（javacコマンドなど）
    CosmoCodeというGUIベースの開発ツールもあるようです。（コマンドはcosmocode）
  • Perl言語
    version 5 のもの（jperl 5.004_04）が使えます。コマンドは、/usr/local/bin/perlで使って下さい。
  • Tcl/Tk 言語
    コマンドは、tclsh, wish などです。 ※ Tcl/Tkは日本語されていません。

応用

様々なアプリケーション・ソフトウェアが、入っています。以下、コマンド名と説明です。
photoshop
画像処理ソフトウェア。Adobe Photoshop 3.0.1
illustrator
描画ソフトウェア。Adobe Illustrator 5.5
premiere
動画処理ソフトウェア。Adobe Premiere 4.2
acroread
PDF文書作成ソフトウェア。Adobe Acrobat Reader 3.0
cosmocreate
統合Webオーサリングソフトウェア。
cosmocode
Java開発環境（JDK1.1.2）。
cosmoworlds
VRML 構築ソフトウェア。
その他もろもろ。

注意点

  • パスワードの変更には、yppasswd コマンドを使用して下さい。
  • 一部に（特にデモ）一般的な X 環境でサポートされていない機能を用いているものがありますので、コンソール以外の X 端末などの上では動かないアプリケーションがあります。（「DOOM」は動かなかった）

現在までに判明している問題点

  • 極端にマシン性能が遅くなる時がある。（メモリ等の調整か？）
  • lp でテキストファイルを印刷すると日本語が完全に文字化けして出力される。
